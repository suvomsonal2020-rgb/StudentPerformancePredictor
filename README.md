# Student Performance Predictor

## Overview

This machine learning project predicts a student's math score based on demographic and academic factors.

## Features

- Predicts math scores
- Interactive Streamlit interface
- Random Forest Regression model
- Real-world dataset

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Model

Random Forest Regressor
